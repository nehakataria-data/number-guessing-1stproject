# number-guessing-1stproject
🎯 Number Guessing Game (Python)

A beginner-friendly Python project where the computer selects a secret number and the player attempts to guess it. The program provides hints after each incorrect guess, telling the user whether the secret number is higher or lower, and continues until the correct number is guessed.

🚀 Features
- Randomly generates a secret number.
- Accepts user input from the keyboard.
- Uses a while loop to keep the game running until the correct guess.
- Provides hints ("Higher" or "Lower") after each incorrect guess.
- Demonstrates the use of variables, conditional statements (if, elif, else), loops, and user input.
🛠️ Technologies Used
Python
📚 Concepts Practiced
- Variables
- User Input (input())
- Data Type Conversion (int())
- Conditional Statements (if, elif, else)
- while Loops
- Random Number Generation (random module)
🎯 Purpose
This project was built to strengthen fundamental Python programming skills by creating an interactive console game while practicing core programming concepts.
